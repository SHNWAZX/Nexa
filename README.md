# Nexa

Nexa is an Android video and audio downloader based on the open source
[Seal](https://github.com/JunkFood02/Seal) project.

## Download

APK builds are published on the
[Nexa releases page](https://github.com/SHNWAZX/Nexa/releases).

For most Android phones, install the `arm64-v8a` APK. If you are not sure which
CPU your device uses, install the `universal` APK.
